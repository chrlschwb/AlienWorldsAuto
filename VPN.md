## How to set up ProtonVPN for this bot
1. Create an account [here](https://account.protonvpn.com/signup/account), select the free plan.
2. Download the client [here](https://protonvpn.com/download)
3. Install
4. Log in
5. ![image](https://user-images.githubusercontent.com/82218562/114304169-115c3f80-9ad2-11eb-99cf-1784ab79dd5f.png) Go in the settings
6. ![image](https://user-images.githubusercontent.com/82218562/114304178-20db8880-9ad2-11eb-8723-84934988d222.png) Enable Split Tunneling for Brave
7. Connect to a server
8. Enjoy!
